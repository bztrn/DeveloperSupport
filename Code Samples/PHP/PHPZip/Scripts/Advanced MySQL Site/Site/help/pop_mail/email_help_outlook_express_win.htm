POP3 EMAIL CONFIGURATION - OUTLOOK EXPRESS / WINDOWS
These directions should work for most recent Microsoft Outlook Express versions, running on Microsoft Windows 98 and later.
Step by Step Connection Help
  • Launch Outlook Express

    Switch Identity (Command-Shift-Q).
  • Click "Switch".
  • Click "New".
  • In the new Identity window enter the name you would like to call this Identity, and base it on the main Identity if you have the choice.
    • -->
  • Under "Tools" select "accounts..."

  • Select the "mail" tab.

  • If there are no accounts present, Please go to the next section of this FAQ called Connection Wizard.

  • If there is already an account and you are ready to change your email address, continue reading this section

  • Select the account by single-clicking on it.

  • Click the "Properties" button.

  • Under Mail Account, type in the name you would like to reffer to this account by.

  • Name: The name you would like displayed
    (i.e. John Doe or ACME Sales)

  • Email address: Alias-you-chose@your-domain-name.com
    (i.e. john@thedoes.com or sales@acme.com)

  • Click the "Servers" tab and input these settings:

  • Incoming email server type: POP (not IMAP)

  • Incoming mail (POP) server: pop.your-domain-name.com
    (i.e. pop.thedoes.com or pop.acme.com)

  • Outgoing mail (SMTP) server: smtp.your-domain-name.com
    (i.e. smtp.thedoes.com or smtp.acme.com)

  • Account ID: Your full email address: alias-you-chose@your-domain-name.com
    (i.e. john@thedoes.com or sales@acme.com)

  • Password: password-you-chose (case sensitive)

  • Check the "My server requires authentication" box and click the "settings" button.
    Make sure the "Use same settings as my incoming mail server" radio button is selected.
    Click "OK".

  • Press "OK"

  • Press "Close"
Settings
  • From the "Mail" tab, click on the "add..." button, and select "Mail".

  • Select the "mail" tab.

  • Input these settings:

  • Display Name: Your name
    (i.e. John Doe or ACME Sales)

  • Email address: Alias-you-chose@your-domain-name.com
    (i.e. john@thedoes.com or sales@acme.com)

  • Incoming email server type: POP (not IMAP)

  • Incoming mail (POP) server: pop.your-domain-name.com
    (i.e. pop.thedoes.com or pop.acme.com)

  • Outgoing mail (SMTP) server: use the current settings from your ISP
    (i.e. mail.attbi.com or mail.msn.com)

  • Account ID: alias-you-chose@your-domain-name.com
    (i.e. john@thedoes.com or sales@acme.com)

  • Password: password-you-chose (case sensitive. This is optional, if you don't want to have to type in this password everytime you send or receive mail)

  • Account Name: Enter what you would like to identify this account
    (i.e. thedoes.com POP mail or acme.com POP mail)

  • Outgoing mail server authentication is required. Same settings as above for incoming mail server (POP).

  • Click "Finish".
